## Getting Started <a name = "gs"></a>

To get a local copy of the repository please run the following commands on your terminal:

```
$ cd <folder>
```

~~~bash
$ git clone git@github.com:Dipeshtwis/ror-social-scaffold.git
$ cd ror-social-scaffold
$ bundle install
$ rails db:migrate
~~~
